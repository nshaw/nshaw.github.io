Get Started
with Entando in
2 Easy Steps

New to Kubernetes and hypervisors? Check out our in-depth guide where you’ll get hands-on experience, and learn Kubernetes as you go for each step of the process.

Install Multipass

Install Multipass as a quick way to setup an Ubuntu VM

https://multipass.run/#install

Install Entando

Install Entando into Kubernetes on Ubuntu using the Entando CLI

curl -sfL https://get.entando.org | bash

The progress of the install will be displayed on the console and can take 10 minutes or so depending on the time needed to download the Docker images.

Once complete, the installer will give you the URL to access the Entando App Builder.

Login with username: admin and password: adminadmin

Choose an action from the Next Steps list to continue your journey with Entando!
